# Language Translator App

A simple and user-friendly Android app to translate text, speech, and images into different languages.  
Built with Kotlin, MVVM architecture, Firebase ML Kit, and Google Gemini API.

---

## Features

- Text Translation – Translate typed text instantly  
- Speech Translation – Speak and get translations in real time  
- Image Translation – Extract and translate text from images  
- Text-to-Speech – Listen to both original and translated text  
- Multi-language Support – Select source and target languages easily  
- Clean UI – Bottom navigation for quick access to features  

---

## Tech Stack

- Language: Kotlin  
- Architecture: MVVM  
- APIs: Google Gemini API, Firebase ML Kit  
- UI: XML  

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AbheenavSingh/Language_Translator.git
````

2. Open the project in **Android Studio**
3. Add your **Firebase configuration file** (`google-services.json`) under the `app/` folder
4. Add your **Gemini API key** in `local.properties`
5. Build and run on an emulator or physical device

---

## Usage

1. Launch the app
2. Select source and target languages
3. Choose a translation method:

   * Text – type manually
   * Image – upload an image
   * Speech – speak directly
4. View the translated output and listen with text-to-speech
